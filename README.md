# Data Cleaning & Visualization Project

Work on a raw dataset to clean, process, and visualize insights.

## Key Features
*   **Handle missing values, outliers, and duplicates:** Cleans raw e-commerce sales datasets, resolving duplicates, managing custom imputations, and pruning outliers using the IQR method.
*   **Use Python libraries like Pandas, Matplotlib, or Seaborn:** Built using Python, Pandas, NumPy, and Matplotlib.
*   **Create dashboards or visual reports of key findings:** Automatically generates 11 analytical plots, an executive infographic dashboard (`output/dashboard.png`), and a detailed summary text report (`output/summary_report.txt`).

## Expected Outcome
*   Learn data preprocessing, visualization, and storytelling with data.
